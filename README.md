# jewellery_app_ai

This repository contains the AI-powered jewellery app.

## Features
- Virtual Necklace Try-on Prototype (Web)
- Flutter User Application

## Setup
- For the Web prototype, run a local server (e.g., `python -m http.server`) and open `index.html`.
- For the Flutter app, run `flutter pub get` in the `jewelley---user-main` directory.
